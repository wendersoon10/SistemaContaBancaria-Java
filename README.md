## Sistema Bancário em Java

Projeto desenvolvido em Java utilizando os conceitos de Programação Orientada a Objetos (POO), simulando um sistema bancário simples.

O sistema possui sobrecarga de construtores, permitindo a criação de contas com ou sem saldo inicial, oferecendo flexibilidade na abertura da conta.

Foram implementadas regras de negócio como:
- Depósito
- Saque com cobrança de taxa fixa de R$ 5,00
- Transferência entre contas

O projeto também utiliza sobrescrita de métodos (`@Override`) para personalizar comportamentos herdados, reforçando conceitos como encapsulamento, abstração e reutilização de código.
